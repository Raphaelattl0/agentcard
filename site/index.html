How it works

From REST API to discoverable agent

No SDK to learn. No server to run. The whole pipeline is one POST request.

01

Send your spec

POST your OpenAPI document, or describe a handful of endpoints by hand if you don't have one yet.

02

We generate & sign

Each operation becomes an A2A skill with a routable description. The card is canonicalized and signed with a dedicated ES256 key.

03

Agents call it today

Your agent is live at a hosted JSON-RPC endpoint. Any A2A-compliant orchestrator can discover and invoke it immediately.
